# Docker
```bash
cd docker

# Build
docker-compose build

# Running without debug
docker-compose up -d
# Running with debug
docker-compose up

# Stop
docker-compose stop
# Remove and stop
docker-compose down

# List all docker images running
docker ps

# Access to a docker image
# e.g
docker exec -it dym_app bash

```
